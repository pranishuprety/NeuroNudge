Parent Mode keeps habits healthy

Set clear guardrails for the sites that drain focus. Parent Mode lives entirely on the device, giving families Chromebook-ready controls without new accounts or dashboards. Keep social tabs closed during class and let curiosity roam again once the school day wraps.

Block the big distractions

Choose sites that stay off-limits while Parent Mode is on. NeuroNudge gently redirects to a calming reminder page if they pop up—ideal for shared family laptops and managed school Chromebooks.

Time-box screen time

Add daily caps or per-session timers for specific sites. When the limit is reached, the browser pauses that destination until it resets, encouraging breaks, stretches, and real-world conversation.

See rules at a glance

Manage everything inside the popup. Toggle Parent Mode, review blocked sites, tweak limits, and wire up alert texts in seconds.

Install NeuroNudge today

While we finish Chrome Web Store review, you can sideload the extension in developer mode. Expect the store listing soon!

  1. Download or clone the project from GitHub.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode in the top right.
  4. Choose Load unpacked and select the downloaded NeuroNudge folder.
  5. Pin the extension and start exploring mindful productivity nudges.
